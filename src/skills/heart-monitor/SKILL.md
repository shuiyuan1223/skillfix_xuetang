# 心率监测技能

当用户询问心率相关问题时使用此技能。

## 使用流程

1. **获取数据**：调用 `get_heart_rate` 获取心率数据
2. **分析心率**：评估静息心率、变化趋势
3. **健康评估**：判断心率是否在正常范围
4. **给出建议**：提供心血管健康建议

## 心率参考值

| 类型 | 正常范围 | 说明 |
|------|----------|------|
| 静息心率 | 60-100 bpm | 成人正常范围 |
| 运动员静息 | 40-60 bpm | 心肺功能好的表现 |
| 最大心率 | 220-年龄 | 理论最大心率 |

## 异常识别

- **静息心率 > 100**：心动过速，建议关注
- **静息心率 < 50**：除非是运动员，否则需注意
- **心率波动大**：可能与压力、咖啡因有关

## 回复示例

用户问："我的心率正常吗？"

1. 调用 `get_heart_rate(date: "today")`
2. 分析返回的数据
3. 回复：
   - 静息心率：X bpm（正常/偏高/偏低）
   - 今日范围：最低X - 最高Y
   - 健康建议

## 注意事项

- 心率数据仅供参考，不构成医疗建议
- 如有持续异常，建议就医检查
