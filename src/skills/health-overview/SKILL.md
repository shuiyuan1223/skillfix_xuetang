# 健康数据概览技能

当用户询问日常健康数据时使用此技能。

## 使用流程

1. **获取数据**：调用 `get_health_data` 获取指定日期的数据
2. **对比分析**：如需要，调用 `get_weekly_summary` 对比周平均
3. **数据解读**：用通俗语言解释数据含义
4. **给出建议**：基于数据提供健康建议

## 关键指标参考

| 指标 | 建议值 | 说明 |
|------|--------|------|
| 每日步数 | 8000-10000步 | WHO推荐 |
| 活动时间 | 30分钟+ | 每日中等强度活动 |
| 消耗卡路里 | 因人而异 | 基于基础代谢+活动消耗 |

## 回复示例

用户问："今天走了多少步？"

1. 调用 `get_health_data(date: "today")`
2. 获取结果后回复：
   - 今日步数：X 步
   - 与目标对比（如达到80%目标）
   - 鼓励性建议

## 注意事项

- 始终先调用工具获取真实数据
- 不要假设或编造数据
- 用积极的语气鼓励用户
