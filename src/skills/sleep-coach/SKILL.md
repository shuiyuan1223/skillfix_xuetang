# 睡眠教练技能

当用户询问睡眠相关问题时，使用此技能提供专业的睡眠分析和建议。

## 分析流程

1. **获取数据**：调用 `get_sleep` 获取最近的睡眠数据
2. **趋势分析**：如需要，调用 `get_weekly_summary` 查看周趋势
3. **质量评估**：分析睡眠时长、深睡比例、睡眠效率
4. **给出建议**：基于数据提供具体可行的改善建议

## 关键指标解读

| 指标 | 理想范围 | 说明 |
|------|----------|------|
| 总时长 | 7-9小时 | 成人推荐睡眠时长 |
| 深睡占比 | 15-25% | 身体恢复的关键阶段 |
| REM占比 | 20-25% | 记忆巩固和情绪调节 |
| 睡眠效率 | >85% | 躺床时间中实际睡眠的比例 |

## 常见问题模板

### 入睡困难
- 检查睡眠时间规律性
- 建议：固定作息、睡前放松、避免蓝光

### 早醒
- 检查深睡和 REM 分布
- 建议：调整卧室环境、检查压力源

### 日间疲劳
- 检查睡眠质量评分
- 建议：增加深睡、检查睡眠呼吸情况
