# PHA 心跳任务

> 心跳任务是 Agent 定期自动执行的后台逻辑。当前 PHA 暂未启用定时心跳，本文件为未来扩展预留。

## 计划中的心跳任务

### 每日健康摘要
- **频率**: 每天早上 8:00
- **动作**: 自动获取昨日健康数据，生成摘要，推送给用户
- **状态**: 未实现

### 异常检测
- **频率**: 每 4 小时
- **动作**: 检查最近健康数据是否有异常模式（心率骤升、睡眠中断等）
- **状态**: 未实现

### 目标追踪提醒
- **频率**: 每天下午 6:00
- **动作**: 检查用户当日步数/运动目标完成情况，未达标时温和提醒
- **状态**: 未实现

### 记忆整理
- **频率**: 每周日
- **动作**: 整理本周对话记忆，归纳关键发现，更新 MEMORY.md
- **状态**: 未实现
