# PHA 工具清单

## 健康数据工具

| 工具 | 说明 |
|---|---|
| `get_health_data` | 获取指定日期的综合健康数据（步数、卡路里、活动量） |
| `get_heart_rate` | 获取心率数据（静息心率、最高/最低、心率曲线） |
| `get_sleep` | 获取睡眠数据（时长、阶段分布、质量评分） |
| `get_workouts` | 获取运动记录（类型、时长、心率区间） |
| `get_weekly_summary` | 获取一周健康概览 |
| `get_stress` | 获取压力数据 |
| `get_spo2` | 获取血氧饱和度 |
| `get_health_trends` | 获取长期趋势（7d/30d/90d/180d/365d） |
| `get_blood_pressure` | 获取血压数据 |
| `get_blood_glucose` | 获取血糖数据 |
| `get_body_composition` | 获取体成分数据 |
| `get_body_temperature` | 获取体温数据 |
| `get_nutrition` | 获取营养摄入数据 |
| `get_menstrual_cycle` | 获取月经周期数据 |
| `get_vo2max` | 获取最大摄氧量 |
| `get_emotion` | 获取情绪数据 |
| `get_hrv` | 获取心率变异性数据 |

## 记忆工具

| 工具 | 说明 |
|---|---|
| `memory_search` | 搜索用户长期记忆 |
| `memory_save` | 保存重要信息到 MEMORY.md |
| `daily_log` | 记录当日对话要点 |

## 用户档案工具

| 工具 | 说明 |
|---|---|
| `update_user_profile` | 保存用户个人信息到 USER.md |
| `complete_onboarding` | 完成首次对话引导，删除 BOOTSTRAP.md |

## 技能工具

| 工具 | 说明 |
|---|---|
| `get_skill` | 按需加载专业技能指南 |

## 配置工具

| 工具 | 说明 |
|---|---|
| `get_config` | 读取系统配置 |
| `update_config` | 更新系统配置 |
| `list_providers` | 列出可用 LLM 提供商 |
